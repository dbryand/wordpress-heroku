Shortcodes Ultimate: Maker Addon
==================================

Provides UI for creating custom shortcodes. [Visit project page](http://gndev.info/shortcodes-ultimate/maker/).

Requirements
------------

* PHP 5.1 or higher
* WordPress 3.5 or higher
* Shortcodes Ultimate 4.4 or higher
